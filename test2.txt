ls:
  -a 列出所有目录，包括隐藏的文件和文件夹
  -l 输出文件名的同时输出权限，所有者，文件大小等详细信息
  -d 将目录像文件一样显示，而不是显示其下的文件
  -h 容易理解饿的格式列出文件大小
  -t 修改时间排序
pwd:
  -P 实际物理路径，而非连接路径
  -L 如果目录为连接路径，显示连接路径
mkdir:
  -m 设定权限模式，再-m后面加数字决定初始权限
  -p 如果建立时有原本不存在的路径，就会自动创建这些路径
  -v 每次创建新目录的时候显示信息

tree:树形结构显示文件目录

rm:
  -f 忽略不存在的文件，不给出提示
  -i 交互模式，每次删除前先进行询问
  -r 递归删除，完全删除
  -v 详细的显示进行的步骤
mv:
  -b 若需覆盖文件，覆盖前先行备份
  -f 若目标文件已经存在，会不进行询问直接覆盖
  -i 若目标文件已经存在，会进行询问再覆盖
  -u 若目标文件已经存在，且源文件比较新，才会更新
  -t 移动多个源文件到一个目录是的情况，此时目标目录在前，源文件在后
cp:
  -t 指定目标目录
  -i 覆盖前询问（使前面的-n选项失效）
  -n 不覆盖已存在的文件（使前面的-i选项失效）
  -s 对源文件建立符号链接，而非复制文件
  -f 强制复制文件或目录，无论文件或目录是否存在
  -u 只会对修改时间较目的文件更新或目的稳健不存在时，会复制文件
cat:
  -b 非空行编号输出
  -E 每行结束显示$
  -n 全部行编号输出
  -s 多个空白行代换为一个空白行
  -T 将跳格字符显示为^|
nl:
  -b a 如果有空行，也同样输出行号
  -b t 如果有空行，空行不输出行号
  -n ln 行号再屏幕最左方显示
  -n rn 行号在自己栏位最右方显示，且不加0
  -n rz 行号在自己栏位最右方显示，且加0
  -w  行号栏位占用的位数（后加数字）
more:
  +n 从第n行开始显示
  -n 定义屏幕大小为n行
  -c 从顶部清屏，然后显示
  -d 提示空格继续Q退出，禁用响铃功能
  -p 通过清除窗口而不是滚屏事先换页
  -s 多个空行只显示一个空行
  -u 把文件内容的下划线去掉
less:
  -e 文件显示结束自动离开
  -f 强迫打开特殊文件
  -i 搜索时忽略大小写
  -m 显示类似more命令的百分比
  -N 显示每行的行号
  -s 显示连续空行为一行
head:
  -q 隐藏文件名
  -v 显示文件名
  -c<字节> 显示字节数
  -n<行数> 显示的行数
tail:
  -f 循环读取
  -q 不显示处理信息
  -v 显示详细的处理信息
  -c<字节> 显示的字节数
  -n<行数> 显示行数
